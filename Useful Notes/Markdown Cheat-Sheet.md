| Style                  | Syntax                                                 | Output                                                                                |
| ---------------------- | ------------------------------------------------------ | ------------------------------------------------------------------------------------- |
| Heading                | H1: `#` <br>H2: `##`<br>H3: `###`                      | # A first-level heading<br>## A second-level heading<br>### A third-level heading<br> |
| Bold                   | `**  **`<br>or<br>`__ __`                              |                                                                                       |
| Italic                 | `*  *`<br>or<br>`_  _`                                 |                                                                                       |
| Strikethrough          | `~~  ~~`                                               |                                                                                       |
| Bold and nested italic | `** **`<br>and<br>`_ _`                                |                                                                                       |
| All bold and italic    | `***  ***`                                             |                                                                                       |
| Subscript              | `<sub>  </sub>`                                        |                                                                                       |
| Superscript            | `<sup>  </sup>`                                        |                                                                                       |
| Blockquote             | `>`                                                    |                                                                                       |
| Ordered List           | `1.`<br>`2.`                                           |                                                                                       |
| Unordered List         | `-  `<br>`-  `                                         |                                                                                       |
| Code                   | `` `  ` ``                                             |                                                                                       |
| Horizontal Rule        | `---`                                                  |                                                                                       |
| Link                   | `[title](Address)`                                     |                                                                                       |
| Image                  | `![alt text](image.jpg)`                               |                                                                                       |
| Fence Code Block       | ` ```     ``` `                                        |                                                                                       |
| Footnote               | `Sentence [^1]`<br><br>`[^1]: Footnote`                |                                                                                       |
| Highlight              | `===   ===`                                            |                                                                                       |
| Table                  | `\|text\|text2\|`<br>`-----------`<br>`\|row1\|col2\|` |                                                                                       |
